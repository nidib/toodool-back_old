# Toodool |BACKEND|
Backend of a sample full stack to-do app - written in node.js

### Instructions for cloning
- Clone this repo
- `$ npm install`
- `$ docker-compose up -d`
- Run the content of `tables.sql` in the just created database
- `$ npm start`

### Project info
Server Language:
- Javascript with node.js

Database:
- PostgreSQL
